# ff14-bot-pricecheack

用來查詢《Final Fantasy XIV》繁中服各伺服器交易板價格的 Discord Bot。

目前支援查詢繁中服各個世界的前 5 筆掛單價格，適合快速比較各伺服器物價。不過部分流動率高的物品，可能因資料來源更新延遲而無法即時反映最新價格。

## 功能

- 支援 Discord 內直接標註 Bot 查詢物品價格
- 支援繁中物品名稱查詢
- 支援英文物品名稱查詢
- 查詢繁中服多個世界價格：
  - 巴哈姆特
  - 鳳凰
  - 泰坦
  - 伊弗利特
  - 迦樓羅
  - 利維坦
  - 奧汀
- 每個世界顯示前 5 筆交易板掛單
- 顯示單價、數量與 HQ 狀態
- 查詢逾時時會提示對應世界查詢失敗或逾時
- 支援 Discord 訊息長度切分輸出
